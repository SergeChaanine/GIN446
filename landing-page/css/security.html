Overview

Security on the Web covers threats that range from opportunistic phishing to sophisticated software supply-chain compromises. Threats affect everyone; users, small businesses, large platforms, and often exploit human factors as well as technical weaknesses.

The internet has transformed communication, business, and daily life, but its openness also introduces risks. Individuals and organizations face threats ranging from phishing to ransomware. By understanding these risks and adopting safe practices, users can reduce exposure and safeguard digital assets.

Common Risks

  • Phishing: Deceptive emails and websites impersonating trusted sources.
  • Malware & Ransomware: Malicious programs that steal, spy, or lock data until payment is made.
  • Data Breaches: Exposed databases leaking sensitive information for misuse.
  • Weak Credentials: Simple or reused passwords enabling unauthorized access.
  • Public Wi-Fi Risks: Attackers intercept data on unsecured networks.
  • Social Engineering: Psychological manipulation that exploits human trust.

Safe Practices

Before entering sensitive information into a website, look for the security padlock symbol. The padlock is one way of checking that a website is safe and indicates that the information you submit is secure.

How to verify website security:

  • Check the padlock icon: Double-click the icon. It will display the certificate information for the page you are viewing to guarantee that you are on a safe, secure website. Make sure the certificate is current and issued to the same company you are visiting.
  • Beware of fake padlocks: Don't be fooled by a padlock that appears on the web page itself because it is possible for a cyber criminal to copy the image. So, double check that the padlock is in the window frame of the browser itself.
  • Verify HTTPS: After validating the padlock, you can also check the text before the website name in the address bar. The "https" is another indication that the page you are viewing is secure.
  • Monitor URL changes: Pay attention to the web address. Check the address and if you click on a link, look to make sure that the address stayed the same. If it has changed, then it has taken you to a fraudulent web address where cyber criminals can monitor and access your information.

Additional Security Tips

  • Use strong, unique passwords for each online account
  • Enable two-factor authentication wherever possible
  • Keep your software and browsers up to date
  • Be cautious with email attachments and links from unknown sources
  • Use a VPN when connecting to public Wi-Fi networks
  • Regularly backup important data
